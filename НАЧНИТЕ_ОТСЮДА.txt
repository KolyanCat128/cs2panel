│ ВАРИАНТ 1: Использовать ПРЯМО СЕЙЧАС (без GitHub)            │
└───────────────────────────────────────────────────────────────┘

  📖 Откройте: КАК_ЗАПУСТИТЬ_СЕЙЧАС.txt

  Кратко:
  1. Скопируйте scripts/install-hypervisor.sh на Ubuntu VM
  2. chmod +x install-hypervisor.sh
  3. sudo ./install-hypervisor.sh
  4. Дождитесь 10-30 минут
  5. sudo reboot
  6. Готово! ✅


┌───────────────────────────────────────────────────────────────┐

  📖 Откройте: ДЛЯ_KOLYANCAT128.txt

  Кратко:
  1. git init
  2. git add .
  3. git commit -m "Initial commit"
  4. Создайте репо на GitHub.com: cs2panel
  5. git remote add origin https://github.com/KolyanCat128/cs2panel.git
  6. git push -u origin main

  После этого ЛЮБОЙ сможет установить одной командой:

  curl -fsSL https://raw.githubusercontent.com/KolyanCat128/cs2panel/main/scripts/install-hypervisor.sh | sudo bash
